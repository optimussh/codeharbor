PARTK 2. 재현해야 할 핵심컨셉 정의
제로베이스로 다시 만들때 꼭 재현해야하는것
1.자연어>에이전트>실제 산출물 : 사람이 말로 시키면 AI 에이전트가 파일 코드 앱을 만든다.
2. 채팅 UX + 실시간 스트리밍(SSE) : 답변 툴 실행이 실시간으로 흐른다
3. 허락(PERMISSION) 게이트 : 위험한 작업은 사람이 승인
4. 멀티유저 격리 : 여러명이 써도 파일 대화 비밀이 절대 안섞인다. (OS 수준 격리)
5. 폐쇄망 RAG +문서 첨부 : 사내 문서를 올려 근거로 활용 (임베딩 ,검색, 재랭킹)
6. 거버넌스된 앱 배포 : 만든 앱을 인증된 사내 URL에 올려 동료가 실제 사용
7. 인증 감사 : 사내 계정 로그인 +접근 기록/감사
8. 폐쇄망(에어갭)운영 : 인터넷 없이 사내 미러 레지스트리로만 동작

선택 (있으면 좋지만 MVP 에는 생략 가능)

C1. AI 에이전트 엔진 가장 중요한 결정
기능 : 자연어 지시로 파일 읽기/쓰기/편집, 셸 명령 실행, 툴(MCP)호출, 멀티턴 대화, 세션/메시지 상태관리., 이벤트 스트리밍(SSE)

최소스펙 : 세션 CRUD 메시지 파트 스트리밍(SSE), 툴 호출 프로토콜 (권장 MCP 표준), 권한 훅, 작업 디텍토리 지정(멀티유저 격리의 전제)
